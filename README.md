# Certificate-Generator-Project
### Project Overview

This project is an automated tool designed to generate
personalized PDF certificates from an Excel dataset.
It was built to solve the manual overhead of creating
certificates for large groups of students, ensuring data
consistency and professional formatting.
<hr>

### Tools & Technologies
* Language: Python
* Libraries: `Pandas` for data cleaning and manipulation
* ReportLab: For high-quality PDF generation and canvas drawing
* Environment: Google Colab
